## 데이터 결합

1. 열 추가

데이터를 가로로 추가하는 것(열 추가)은 'merge' 함수를 사용한다.

~~~python
test1 = pd.DataFrame({'id' : [1, 2, 3, 4, 5], 'midterm' : [60, 80, 70, 90, 85]})
test2 = pd.DataFrame({'id' : [1, 2, 3, 4, 5], 'midterm' : [70, 83, 65, 95, 80]})

total = pd.merge(test1, test2, how = 'left', on = 'id')
~~~
